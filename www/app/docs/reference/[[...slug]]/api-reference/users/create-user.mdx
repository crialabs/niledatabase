# Create a User

<div className="grid grid-cols-2 gap-4">

    <p>The response includes the user id and an access token for the user. The access token can be used to perform API operations on the user's behalf.</p>
    <p>You can also create a user and a new tenant for that user in a single request by specifying the `newTenant` attribute.</p>
    <p>The response includes the user id and an access token, and the `tenants` array includes the id of the new tenant.</p>
